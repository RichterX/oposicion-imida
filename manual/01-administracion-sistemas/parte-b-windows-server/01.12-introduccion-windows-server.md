# ¿Qué es Windows Server?

Windows Server es la familia de sistemas operativos de Microsoft diseñada específicamente para ejecutar servicios de red, aplicaciones empresariales y administrar infraestructuras informáticas.

A diferencia de Windows 10 o Windows 11, no está orientado al usuario final, sino a proporcionar servicios a otros equipos.

Entre esos servicios se encuentran:

- autenticación de usuarios;
- compartición de archivos;
- servidores web;
- virtualización;
- bases de datos;
- administración de redes.

---

# ¿Por qué existe Windows Server?

Un ordenador personal suele dar servicio a un único usuario.

Un servidor, en cambio, puede atender simultáneamente a cientos o miles de usuarios.

Windows Server incorpora herramientas específicas para:

- administración centralizada;
- alta disponibilidad;
- seguridad;
- virtualización;
- gestión de identidades;
- almacenamiento.

---

# Windows cliente vs Windows Server

| Windows 11 | Windows Server |
|------------|----------------|
| Uso personal | Uso empresarial |
| Un usuario principal | Múltiples usuarios y servicios |
| Aplicaciones de escritorio | Servicios de red |
| Productividad | Administración de infraestructura |

🎯 **Idea clave**

Windows Server no sustituye a Windows 11.

Cumple un propósito completamente diferente.

# LTSC

Microsoft publica versiones de soporte a largo plazo (LTSC).

Estas versiones reciben actualizaciones de seguridad durante muchos años y son las preferidas en entornos empresariales.

---

# Modalidades de instalación

Windows Server puede instalarse de dos formas.

## Desktop Experience

Incluye interfaz gráfica.

Es la modalidad más utilizada para aprender y administrar servidores.

---

## Server Core

No dispone de escritorio tradicional.

Se administra mediante:

- PowerShell;
- Windows Admin Center;
- herramientas remotas.

Consume menos recursos y reduce la superficie de ataque.

---

# 🐧 Linux vs 🪟 Windows

| GNU/Linux | Windows Server |
|------------|----------------|
| Bash | PowerShell |
| root | Administrator |
| systemd | Service Control Manager |
| ext4 | NTFS |
| SSH | RDP / PowerShell Remoting |

🎯 Ambos sistemas persiguen el mismo objetivo: administrar servicios para otros equipos.

---

# 💻 Windows en acción

Abres el **Administrador del servidor (Server Manager)**.

Desde una única consola puedes:

- instalar roles;
- administrar discos;
- configurar red;
- revisar eventos;
- agregar servidores remotos.

Es el equivalente al "centro de operaciones" de un administrador Windows.

---

# 🖥️ Laboratorio recomendado

En una máquina virtual:

1. Instala Windows Server 2022.
2. Selecciona la opción **Desktop Experience**.
3. Inicia sesión como Administrator.
4. Abre Server Manager.
5. Identifica:
   - Nombre del servidor.
   - Dirección IP.
   - Estado de Windows Update.
   - Roles instalados.

No modifiques todavía ninguna configuración.

Simplemente familiarízate con el entorno.

# ⚠️ Errores frecuentes

❌ Pensar que Windows Server es "un Windows con más potencia".

✔ Es un sistema operativo orientado a proporcionar servicios.

---

❌ Creer que siempre necesita interfaz gráfica.

✔ Puede instalarse como Server Core.

---

❌ Pensar que Windows Server sustituye al equipo del usuario.

✔ Normalmente convive con equipos Windows 10/11 dentro de una misma organización.
